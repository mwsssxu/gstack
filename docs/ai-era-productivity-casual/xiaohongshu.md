## 🤝 协同作战秘籍

### 文档驱动信息传递 📁

```
产品思考者 → 设计文档
战略规划师 → 战略计划
架构设计师 → 架构计划  
偏执专家 → 审查报告
质量专家 → 质量报告
发布专家 → 发布报告
```

**每个角色都知道完整信息，不需要你重新解释！**

### 黄金法则 ✨

1. **文档优先**：永远先创建文档，再开始工作
2. **信息继承**：每个角色都读取之前的文档
3. **质量门禁**：设置标准，确保质量
